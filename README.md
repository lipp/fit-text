# fit-text

A very basic and lightweight fit-text web component (minified + gzip < 500 byte).

# Usage

```html
<script src="https://unpkg.com/fit-text-webcomponent"></script>

<div style="width: 50vw;">
  <fix-text>This fits to 50vw</fit-text>
</div>
```
